Math Fact Invaders — SwiftUI port

Instructions:
- Open Xcode and create a new iOS project using the "App" template and SwiftUI lifecycle.
- Replace or add the Swift files from this folder into the new project (MathFactInvadersApp.swift, ContentView.swift, GameModel.swift, GameScene.swift).
- In the project's target settings, ensure the deployment target is iOS 15.0+.
- Build and run on a simulator or device.

This port uses SwiftUI for UI and SpriteKit for the falling invaders.
